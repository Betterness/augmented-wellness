# Reputation Rules

Protect the operator's reputation. Before a draft is queued it must pass:

- Sounds like the operator; adds something useful; not engagement-bait or a hot take.
- No overclaiming, no dunking, no unverified claims about people or companies.
- No medical advice (see `medical-claims-policy.md`).
- "Would the operator be comfortable defending this publicly?" — if not, don't queue it.
- Discloses AI assistance; never deceptively impersonates.

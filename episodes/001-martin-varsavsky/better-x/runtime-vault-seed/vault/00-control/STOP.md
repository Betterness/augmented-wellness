# STOP

The kill switch. While STATUS is ACTIVE, the agent must take **no external action** — no posting, replying, liking, following, reposting, quoting, DMing, or messaging — regardless of any other setting.

STATUS: ACTIVE

To allow external actions, ALL must be true: (1) STATUS here is INACTIVE, (2) `autonomy.stop_all_external_actions=false` and `x.write_enabled=true` in betterx.config, and (3) an approved publication-gate artifact exists for the exact item. Any one missing = no action.

When in doubt, STOP wins.

# Relationship Index

Status: starter template.

Purpose: give better-x fast relationship context before drafting replies, quote posts, public mentions, or posts about named people and organizations.

## Required Use

Before drafting around a named person or organization:

1. Check this index.
2. Open the linked profile if one exists.
3. Put `relationship_context` in the review artifact.
4. If no match exists, mark the context as `unknown` and avoid fake familiarity.
